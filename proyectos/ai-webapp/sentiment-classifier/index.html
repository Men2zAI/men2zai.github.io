Clasificador de Sentimientos con IA

Esta aplicación web utiliza inteligencia artificial para analizar el sentimiento de textos escritos (positivo, negativo o neutral). Está diseñada para startups, encuestas o análisis de redes sociales.

Características:

  • Modelo ligero de IA integrado en el backend
  • Interfaz simple y responsive
  • Desplegado en la nube o servidor local
